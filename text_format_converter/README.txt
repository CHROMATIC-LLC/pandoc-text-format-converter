Text Format Converter
=====================
Uses Pandoc to convert body text from one format to another.


INSTALLATION
-------------
Install this module as usual. Please see
http://drupal.org/documentation/install/modules-themes/modules-7


USAGE
-------
To use this module, navigate to the administration screen at:
admin/config/content/convert

For more information, see the post Converting Drupal Text Formats with Pandoc at
http://www.chromaticsites.com/blog/converting-drupal-text-formats-pandoc
